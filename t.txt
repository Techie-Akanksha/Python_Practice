This is a great beginner project because you can keep improving it while practicing different Python concepts.

Here are some ideas, roughly from easiest to more advanced:

1. Generate a random number

Instead of always using 123, let the computer choose a number.

    import random

    num = random.randint(1, 100)

Concepts: modules, randomness.

2. Limit the number of attempts

Give the player only 7 tries.

    max_attempts = 7

    while count < max_attempts and num != gnum:
        ...

Concepts: combining conditions.

3. Show remaining attempts
    print(f"Attempts left: {max_attempts - count}")

Concepts: f-strings.

4. Add difficulty levels
    level = input("Choose difficulty (easy/hard): ")

    if level == "easy":
        num = random.randint(1, 50)
    else:
        num = random.randint(1, 500)

Concepts: branching logic.

5. Give warmer/colder hints
    difference = abs(num - gnum)

    if difference <= 5:
        print("Very close!")
    elif difference <= 15:
        print("Close!")
    else:
        print("Far away!")

Concepts: abs(), game feedback.

6. Handle invalid input

Instead of crashing when someone enters text:

    try:
        gnum = int(input("Guess the number: "))
    except ValueError:
        print("Please enter a valid number.")
        continue

Concepts: exception handling.

7. Play again option
    while True:
        # game code

        again = input("Play again? (y/n): ")
        if again.lower() != "y":
            break

Concepts: nested loops.

8. Put the game in functions
    def play_game():
        ...

    play_game()

Concepts: functions, code organization.

9. Track the best score
    best_score = None

    if best_score is None or count < best_score:
        best_score = count

Concepts: state management.

10. Create a computer-guesses-your-number version

The user thinks of a number, and the computer guesses it using binary search.

Example:

Is your number 50?
higher
Is your number 75?
lower
Is your number 62?

Concepts:

Algorithms
Binary search
Efficient problem solving

This is one of the best practice projects after the basic guessing game.
